== Description ==
WishSimply is a simple online wish list / gift registry service that respects users privacy.

This plugin will allow you to map your wish lists to short codes that can be used in Wordpress. Short codes can then be placed on pages or posts where the user want's his/her wish list from wishsimply to come visible.

Plugin development happens in [gitlab](https://gitlab.com/wishsimply/wishsimply-wordpress-plugin/tree/master).

== Installation ==
* Install plugin from the wordpress.org repository (or Upload plugin .zip file to the /wp-content/plugins/ directory and unzip it)
* Activate the plugin through the ‘Plugins’ menu in WordPress.
* Start by visiting WishSimply plugin in WordPress Menu

== Frequently Asked Questions ==

= How WishSimply respects users privacy? = 

WishSimply don't don't give your private information to any 3rd party without your consent. Not even Facebook (through facebook connect) or Google (through google analytics). WishSimply don't show ads or use tracking cookies.  
Read more in WishSimply's [privacy policy](https://wishsimply.com/privacy-policy)

Note: This plugin is for promoting public wish lists, and as the name will tell they are not private.
